MySQL：JDBC 连接数据库

一、JDBC工作原理

JDBC是Java语言中用于与数据库进行交互的一组API（应用程序接口）。JDBC允许Java应用程序通过标准的SQL语句访问数据库，并处理查询、更新和管理数据库中的数据。
JDBC提供了一种统一的方式，使Java应用程序能够与各种不同类型的数据库进行通信，而不需要针对每个数据库系统编写特定的代码。它是一种用于执行SQL语句的Java API，它是 Java中的数据库连接规范。这个API由 java.sql.*，javax.sql.* 包中的一些类和接口组成，它为Java 开发人员操作数据库提供了一个标准的API，可以为多种关系数据库提供统一访问。

SQL 语句本身是 String 类型的，但是 JDBC 提供了 Statement 对象，让我们把 String 转化为 Statement 再发给服务器执行。但是，我们一般使用 PreparedStatement来代替 Statement。
Statement是把 SQL 原封不动的转发给数据库服务器，数据库服务器负责自己解析 SQL，而 PreparedStatement会先在客户端这边初步解析一下 SQL 语句。
